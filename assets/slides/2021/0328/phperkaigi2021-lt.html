Steps

  1. アプリケーションのimage化
    • “localのファイル” からの脱却
  2. image化したアプリケーションを local で動かす

  3. Terraform でAWSにインフラ構築
    • (ECS は クラスター まで）
  4. ecspresso で ECS サービス・タスク定義構築

  5. “だれでも・どこからでもデプロイ” へ簡素化
    • CI/CD 工夫中

特記事項

  • Docker 系の作業は AWS Linux WorkSpaces で
    • Docker for Mac の速度に我慢できない
  • Vue.js の npm run production とのたたかい
    • Laravel Mix の不可解な制約
    • 環境変数を direnv で適用
  • Terraform HCL の Module 化
    • “1日 Production” を実現
  • Fargate 1.4.0 対応
    • PrivateLink を使うには 443 を開ける

  • ブルーグリーンデプロイ

おわりに


  • Fargate化に取り組んだことで意識をサーバーからコンテナに向き変えることが進んだと思う

  • 次から次へと新機能が出てくるのは便利で素晴らしいが、わりと心臓に悪い

  • フリーランスな自分でも、見たエラーをSNSにこぼしては拾ってもらい、課題解決に取り組めていることを大事にして、これからも精進したい
