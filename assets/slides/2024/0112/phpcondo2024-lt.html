定石を捨て、肉付けし直す


  • 「jwt, laravel/passport を上げていく」を断念

    • PHP 8.1, Laravel 9.0 の「骨」をゼロベース作成

  • Laravel 8 でモデルファクトリが激変

    • migration 周りを大幅調整

  • Login 成功を皮切りに、順次 Query, Mutation を移植

うごかないな？を潰していく


  • maatwebsite/excel ライブラリ投入時に前提不足

    • gd を enable（インストールして php.ini 変更）
    • phpoffice/phpspreadsheet を入れる

  • REST API の route の書き方が v7 と v9 で変わってた

  • ValidatorDirective -> Validator に要変更、だった（↓※7）

    • 他にも lighthouse-php v4 -> v5 対応いくつか（↓※8〜9）
  • ※7. Replace custom validation directives with validator classes
  • ※8. Replace @bcrypt with @hash
  • ※9. @orderBy argument renamed to column
