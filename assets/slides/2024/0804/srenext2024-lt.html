Beyond “➕ ➖ ✖️”


  • 共通の課題と向き合う仲間との対話 => Priceless
  • 多様な技術との接点 => 気付きの機会が増えている
  • 標準が把握できる：相場・定石・鉄則・守破離の境界

  • コンディションコントロール難しい => 無理しないで
  • 状況は千変万化 => 終わりを受け入れる準備を
  • 契約や法律の解釈は無限になり得るので十分に注意を
